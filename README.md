## Installation

- Flask@3, Flask-CORS, Flask-SQLAlchemy
- Vue@3, Bootstrap@5.3.3, Popper@2.11.8

```bash
pip install -r requirement.txt
```

## Configuration

- Backend: `config.py`
- Frontend: `vue.config.js`

## Usage

```bash
python backend/app.py
cd frontend && npm run serve
```
